static extract_numeric_values_from_string(str_contains_values)¶

Find numeric values from string, e.g., 1, .7, 1.2, 4e2, 3e-3, -9, etc. reference: https://stackoverflow.com/questions/4703390/

how-to-extract-a-floating-number-from-a-string-in-python/4703508#4703508

Examples

“.1 .12 9.1 98.1 1. 12. 1 12” ==> [0.1, 0.12, 9.1, 98.1, 1.0, 12.0, 1.0, 12.0] “-1 +1 2e9 +2E+09 -2e-9” ==> [-1.0, 1.0, 2000000000.0, 2000000000.0, -2e-09] “current level: -2.03e+99db” ==> [-2.03e+99]

参数:str_contains_values – string which may contains numeric values
返回:list of numeric values
